# 有關董事失格制度

現行法僅有董事消極資格制度（§30），無董事失格制度。未來修法如大幅放寬管制，針對少數強行規範宜引進英美法系失格制度強化監理。董事違反公司法特定規定情節嚴重者，法院或主管機關得為失格處分，禁止其於一定期間內擔任該公司董事。違反失格處分，於一定條件下尚有刑責，且需就違法經營公司期間之公司債務負個人清償責任。以遏阻不適格者管理公司，強化公司治理。
